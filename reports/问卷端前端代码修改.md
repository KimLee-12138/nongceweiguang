下面我为你重构了这几个核心部分的完整代码。你可以直接让 Codex/Cursor 用这些代码替换原文件。

### 1. 核心问卷组件：`ProfileOnboarding.vue`

这个组件完全抛弃了传统的表单和 Element Plus 的默认样式，纯手工打造了卡片流。

- **苹果感**：干净的浅色背景，超大号的标题，平滑的阴影与极简的图标。
- **多邻国感**：使用了 `cubic-bezier` 模拟物理弹簧效果，点击卡片时有明显的下压和回弹反馈（Scale 缩小再放大），单选题自动延迟翻页。
- **深绿色基调**：主色调采用了高级的**森林深绿（#1A4D2E）**。

// 计算属性

const isLastQuestion = computed(() => currentIndex.value === questions.length - 1)
const progressPercentage = computed(() => ((currentIndex.value + 1) / questions.length) * 100)

const canContinue = computed(() => {
  const field = currentQuestion.value.field
  if (field.includes('extra_data.')) {
    const key = field.split('.')[1]
    const val = answers.value.extra_data[key]
    return Array.isArray(val) ? val.length > 0 : !!val
  }
  return !!answers.value[field]
})

// 交互逻辑
const isSelected = (val) => {
  const field = currentQuestion.value.field
  if (field.includes('extra_data.')) {
    const key = field.split('.')[1]
    const currentVal = answers.value.extra_data[key]
    return Array.isArray(currentVal) ? currentVal.includes(val) : currentVal === val
  }
  return answers.value[field] === val
}

const handleSelect = (val) => {
  const q = currentQuestion.value
  const isExtra = q.field.includes('extra_data.')
  const fieldKey = isExtra ? q.field.split('.')[1] : q.field

  if (q.type === 'single') {
    if (isExtra) answers.value.extra_data[fieldKey] = val
    else answers.value[fieldKey] = val
    
    // 多邻国交互核心：单选自动延迟翻页
    setTimeout(() => {
      if (canContinue.value) handleContinue()
    }, 350)
  } else if (q.type === 'multiple') {
    const arr = answers.value.extra_data[fieldKey]
    const idx = arr.indexOf(val)
    if (idx > -1) arr.splice(idx, 1)
    else arr.push(val)
  }
}

const handleContinue = async () => {
  if (!canContinue.value) return
  if (isLastQuestion.value) {
    await submitProfile()
  } else {
    currentIndex.value++
  }
}

const handleExit = () => {
  router.push('/chat') // 或跳回首页
}

const submitProfile = async () => {
  isSubmitting.value = true
  try {
    // 模拟网络延迟以展示骨架屏动画
    await new Promise(res => setTimeout(res, 1500))
    await saveUserProfile(answers.value)
    router.push('/chat')
  } catch (error) {
    console.error('保存画像失败', error)
  } finally {
    isSubmitting.value = false
  }
}
</script>

### 2. 前端路由拦截逻辑 (`router/index.js` 或类似文件)

这段代码的目的是：当用户登录成功后，如果后端标记其为“首次登录”（或者你检测到当前 `userProfiles` 为空），则强制重定向到全屏的问卷页，不允许进入主工作台。

JavaScript

```
import { createRouter, createWebHistory } from 'vue-router'
import { useUserStore } from '@/stores/user' // 假设你使用 Pinia 存储状态

const routes = [
  // ... 其他路由
  {
    path: '/onboarding/profile',
    name: 'ProfileOnboarding',
    component: () => import('@/views/onboarding/ProfileOnboarding.vue'),
    meta: { 
      requiresAuth: true,
      hideLayout: true // 告诉根组件 App.vue 隐藏导航栏和侧边栏
    }
  },
  {
    path: '/chat',
    name: 'Chat',
    component: () => import('@/views/chat/ChatWindow.vue'),
    meta: { requiresAuth: true }
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

router.beforeEach(async (to, from, next) => {
  const userStore = useUserStore()
  
  if (to.meta.requiresAuth && !userStore.isLoggedIn) {
    return next('/login')
  }

  // 初次创建账号拦截逻辑
  if (to.path === '/chat' && userStore.isLoggedIn) {
    // 检查用户是否有画像（可通过调用 API 或读取 store 状态）
    const hasProfile = userStore.profiles && userStore.profiles.length > 0
    // 或者检查后端的 is_first_login 标记
    if (!hasProfile) {
      return next('/onboarding/profile')
    }
  }

  next()
})

export default router
```

### 3. API 请求封装 (`api/profile.js`)

用于提交最终组装好的问卷数据，适配后端结构。

JavaScript

```
import axios from '@/api/client' // 你的 axios 实例

/**
 * 提交并保存用户画像问卷数据
 * @param {Object} data 包含 type, area, green_cert, irrigation, extra_data
 */
export const saveUserProfile = async (data) => {
  try {
    const response = await axios.post('/api/v1/profiles', data)
    return response.data
  } catch (error) {
    console.error('Failed to save profile:', error)
    throw error
  }
}

/**
 * 获取当前用户的画像列表（用于路由拦截判断）
 */
export const getUserProfiles = async () => {
  try {
    const response = await axios.get('/api/v1/profiles')
    return response.data
  } catch (error) {
    console.error('Failed to fetch profiles:', error)
    throw error
  }
}
```
